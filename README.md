## 使用
直接将本目录软链接到 `~/.config/nvim`

## 字体

https://github.com/Karmenzind/monaco-nerd-fonts/blob/master/fonts/Monaco%20Nerd%20Font%20Complete%20Windows%20Compatible.otf

## 替代 vi 和 vim

nvim ~/.zshrc
```
alias vim='nvim'
alias vi='nvim'
alias v='nvim'
```
